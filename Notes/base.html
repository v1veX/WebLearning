Тег "h1" должен быть только ОДИН на всей странице, т.к. это тег ЗАГОЛОВКА СТРАНИЦЫ

Обычно, это тег заголовка секции

Если где-то есть тег h2, то использование после него тега h4 будет ошбкой, поэтому теги нужно использовать строго по порядку

  • Кнопки навигации рекомендуется
  • оборачивать в
  • списки

Выделяет текст (хз зачем, но вещь наверное полезная)

Атрибут "download" позволяет делать ссылки для СКАЧИВАНИЯ ФАЙЛОВ (работает ТОЛЬКО ПРИ ИСПОЛЬЗОВАНИИ СЕРВЕРА) Так пишутся ссылки с электронной почтой А так ссылки с номером телефона (удобно для мобилок, чтобы звонить в один клик) Ссылка откроется в новой вкладке
